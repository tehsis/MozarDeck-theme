Desarrollo para FirefoxOS
y la Web abierta

Objetivos de Firefox OS

  • Llevar la web a un nuevo nivel.
  • Crear un ecosistema abierto para el desarrollo de aplicaciones.
  • Proveer un stack completo para el desarrollo de aplicaciones alcanzando (o superando) el ofrecido actualmente (Android, iOS, Windows Phone, ...) .

Caracteristicas de Firefox OS

  • Aplicaciones desarrolladas usando un stack 100% web.
  • API Basada en estandares (WebAPI).
  • Marketplace completamente descentralizado.

Arquitectura

Tipos de aplicaciones

  • Aplicaciones Alojadas (hosted apps)
  • Aplicaciones empaquetadas (package apps)
    • Se encuentran completamente empaquetadas dentro del dispositivo.
    • Permiten acceder a funcionas críticas del telefono (llamadas, mensajes, ...)
    • Deben estar aprobadas por el marketplace en cuestión.

Cómo se desarrollo una aplicación (Para FirefoxOS).

Tabu

  • Juego de cartas de dos equipos.
  • En cada turno, un jugador debe lograr que su equipo adivine la mayor cantidad de cartas en 1' 30".
  • El jugador no puede usar las palabras (ni sus derivados) que se indican en las cartas.
  • Tampoco puede usar gestos y/o onomatopeyas.

Tabu.js

Una forma de mostrar las cartas y administrar el juego sin necesidad de llevarlas y teniendolas siempre disponibles

  • Fuente: http://github.com/tehsis/tabujs
  • Pruebas: http://tehsis.dlinkddns.com (Completamente inestable)

  • Objetivos

    • Poder jugar sin llevar las cartas.
    • Funcionar en dispositivos (especialmente telefonos) móviles.
    • Poder jugar remotamente.

    • Estructura

    • Servidor

      • Node.js (cón express)
      • MongoDB

    • Cliente

      • Backbone
      • Underscore
      • Zepto
      • Require.js
      • Normalize.css
      • Less
      • + La magia de las webapps...

    Utilizadas

    AppCache

    • Permite acceder de forma offline a la app.
    • Tenemos que asegurarnos de proveer rutas para cada recurso utilizado.

    LocalStorage

    • Permite persistir el estado de la aplicación si la cerramos
    • Al cerrarla durante una partida si la abrimos vamos a tener:
      • El puntaje de cada equipo.
      • Las cartas que habiamos usado.
      • El tiempo transcurrido del turno.

    MozApps

    • Nos permite instalar la app en el dispositivo.
    • Sensación de app nativa (chrome-less)
